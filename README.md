## 概要
アルファベット、数字、記号、スペースの全角文字と半角文字を相互変換するプログラムです。

![picture](http://i.imgur.com/kvckES7.gif)

## 操作方法
- 「←」ボタン(undo)  
直前に実行した操作を取り消します。
- 「→」ボタン(redo)  
undoによって取り消された操作を実行します。
- 「paste」ボタン  
クリップボードに保存されているテキストをテキストエリアに上書きします。
- 「full→half」ボタン  
全角の文字を半角に変換します。
- 「half→full」ボタン  
半角の文字を全角に変換します。
